Overview

One event in a streamed Responses turn (Resources::Responses#stream_create).

The Responses API emits named SSE events; each payload repeats the name in its #type and carries a 0-based #sequence_number. The observed sequence for a simple turn is response.created → response.output_item.added → response.output_text.delta (one per delta) → response.output_text.done → response.output_item.done → response.completed (terminal; there is no [DONE] sentinel). Which readers are meaningful depends on #type; the rest return nil.
